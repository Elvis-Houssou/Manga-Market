    # Toriko

    Titre original : トリコ

    Pays : Japon

    Format : Série TV

    Genre : Action, Aventure, Comédie, Fantasy

    Durée par épisode : 23 min

    Studio d'animation : Toei Animation

    Sous licence : Oui

    Editeur / Fansub : Kazé

    Site web officiel : Lien

    Synopsis :

    L'histoire de Toriko prend place dans un monde où l'art culinaire occupe une place très importante. Toriko est un personnage charismatique de l'ère Gourmet ; il est un célèbre bishokuya, un Chasseur culinaire, réputé pour avoir découvert près de deux pour cent des trois cent mille Ingrédients connus de par le monde. Son but étant de découvrir des ingrédients encore inconnus, il parcourt ce monde peuplé de créatures Plus dangereuses les unes que les autres, accompagné du jeune chef Komatsu. Ainsi, ils vont ensemble affronter puis déguster de nombreuses bêtes savoureuses
